## 状態一覧

- Unchecked / Checked / Hover / Focus / Disabled

### バリアント設定

<DocumentImage src="/img/ui/menu-variant-create.png" alt="バリアント追加" />

- プロパティ名: `State`
- 値: `Unchecked` | `Checked` | `Hover` | `Focus` | `Disabled`

### 視覚調整

- Hover: 外枠の色をわずかに濃く
- Focus: フォーカスリングを追加
- Disabled: 全体の不透明度を 40-60% に、カーソルは not-allowed
