## 状態一覧

- Default / Hover / Focus / Disabled / Loading

### バリアント設定

<DocumentImage src="/img/ui/menu-variant-create.png" alt="バリアント追加" />

- プロパティ名: `State`
- 値: `Default` | `Hover` | `Focus` | `Disabled` | `Loading`

### 視覚調整の目安

- Hover: 背景トーンアップ、カーソル pointer
- Focus: フォーカスリングを明確に（アイコンのみは特に）
- Disabled: テキスト/アイコン不透明度 40-60%、背景/枠線コントラストを弱める
- Loading: スピナー表示、アイコンと置換するか隣接表示
