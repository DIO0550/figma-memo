## 状態一覧

- Default / Placeholder / Focus / Disabled / Error

### バリアント設定

<DocumentImage src="/img/ui/menu-variant-create.png" alt="バリアント追加" />

- プロパティ名: `State`
- 値: `Default` | `Placeholder` | `Focus` | `Disabled` | `Error`

### 視覚調整

- Placeholder: テキストの明度を上げる/不透明度を下げる
- Focus: 枠線色の強調、フォーカスリング
- Disabled: 全体のコントラストを下げ入力不可を示す
- Error: 枠線/ヘルプテキストをエラー色に
