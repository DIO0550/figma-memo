---
title: インタラクションの作成
---

## 概要

Figma でボタンにインタラクションを設定することで、クリック時の動作やページ遷移をプロトタイプで確認できます。ここでは、ボタンクリック時のインタラクション設定方法を説明します。

## 前提条件

- ボタンコンポーネントが作成済みであること
- プロトタイプ用のページ（フレーム）が複数あること

## 基本的なインタラクション設定

### 手順

1. **プロトタイプモードに切り替え**

   - 画面右上の「プロトタイプ」タブをクリックします

   <DocumentImage
     src="/img/button/03/prototype-mode/select.png"
     alt="プロトタイプタブの選択"
   />

2. **ボタンを選択**

   - インタラクションを設定したいボタンを選択します

   <DocumentImage
     src="/img/button/03/button-selection/select.png"
     alt="ボタンの選択状態"
   />

3. **インタラクションを追加**

   - インタラクション項目の右側に表示される「+」ボタンをクリックします

   <DocumentImage
     src="/img/button/03/interaction-add/plus-button.png"
     alt="インタラクション追加ボタン"
   />

- インタラクションが追加されて、インタラクション設定メニューが表示されます
  <DocumentImage
    src="/img/button/03/interaction-add/menu-open.png"
    alt="インタラクション設定メニューの表示"
  />

4. **基本設定**

   インタラクション設定メニューで基本的な動作を設定します：

   - **トリガー**: インタラクションを発動する条件を選択
     - タップ（デフォルト）
     - ドラッグ
     - ホバー
     - キー入力 など
   - **アクション**: 実行する動作の種類を選択
     - Navigate to（ページ遷移）
     - Open overlay（オーバーレイ表示）
     - Back（前のページに戻る）
     - Close overlay（オーバーレイを閉じる）
     - Scroll to（指定位置にスクロール） など

## 画面遷移の設定

### ページ遷移

基本的なボタンクリックでページ間の遷移を設定します。

1. **トリガーの設定**
   - 「クリック時」を選択します
2. **アクションの選択**
   - 「次に移動」を選択します
3. **遷移先フレームを選択**
   - 移動先のページ（フレーム）を指定します

<DocumentImage
  src="/img/button/03/page-transition/setting-complete.png"
  alt="ページ遷移の設定完了画面"
/>

### オーバーレイ表示

ボタンクリックでのモーダルやポップアップを表示する場合：

1. **トリガーの設定**
   - 「クリック時」を選択します
2. **遷移タイプを変更**
   - 「オーバーレイを開く」を選択します
3. **表示位置の調整**
   - 中央、上中央などから選択します
4. **背景設定**
   - 背景の透明度や色を調整します

<DocumentImage
  src="/img/button/03/overlay-display/setting-complete.png"
  alt="オーバーレイ表示の設定完了画面"
/>

## ボタンの状態インタラクション

### ホバー状態

マウスオーバー時のボタンの見た目変化を設定します。

1. **コンポーネントバリアントの作成**

   - Default 状態と Hover 状態のバリアントを作成

2. **Default 状態にインタラクションを設定**

   Default 状態のボタンにホバーインタラクションを追加します：

   - **トリガー**: 「マウスオーバー時」を選択
   - **アクション**: 「次に変更」を選択
   - **プロパティ**: 「Hover」状態を選択
   - **アニメーション**: 任意

<DocumentImage
  src="/img/button/03/hover-state/interaction-setting.png"
  alt="ホバーインタラクションの設定画面"
/>

### プレス状態

クリック時（押下時）のボタンの見た目変化を設定します。

1. **コンポーネントバリアントの作成**

   - Default 状態と Pressed 状態のバリアントを作成

2. **Default 状態にインタラクションを設定**

   Default 状態のボタンにプレスインタラクションを追加します：

   - **トリガー**: 「押下中」を選択
   - **アクション**: 「次に変更」を選択
   - **プロパティ**: 「Pressed」状態を選択
   - **アニメーション**: 任意

<DocumentImage
  src="/img/button/03/press-state/interaction-setting.png"
  alt="プレスインタラクションの設定画面"
/>
