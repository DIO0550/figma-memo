# Card コンポーネントテスト

MDX で Card コンポーネントを表示するテストです。

<Card title="テストカード" description="これはテスト用のカードです。" />

通常のマークダウンテキストもちゃんと表示されます。

## リンク付き Card

<Card
  title="リンク付きカード"
  description="クリックするとリンクに移動します。"
  link="/docs/intro"
/>

## 外部リンク Card

<Card
  title="外部リンクカード"
  description="外部サイトにリンクします。"
  link="https://docusaurus.io"
  external={true}
/>

<Card
  title="横向きカード"
  description="横向きレイアウトのカードです。"
  orientation="horizontal"
  image="/img/docusaurus.png"
  imageAlt="Docusaurus Logo"
/>

## コンポーネントの使い方

上記のカードは以下のように MDX で記述できます：

```jsx
<Card
  title="タイトル"
  description="説明文"
  image="/img/image.png"
  imageAlt="画像の説明"
  link="/docs/link"
/>

<Card
  title="外部リンクカード"
  description="外部サイトへのリンク"
  link="https://example.com"
  external={true}
  size="large"
  orientation="horizontal"
/>
```

## 利用可能なプロパティ

- `title`: カードのタイトル（文字列）
- `description`: カードの説明文（文字列）
- `image`: 画像のパス（文字列）
- `imageAlt`: 画像の代替テキスト（文字列）
- `link`: リンク URL（文字列）
- `external`: 外部リンクかどうか（boolean）
- `size`: 'small' | 'medium' | 'large'
- `orientation`: 'vertical' | 'horizontal'
- `clickable`: クリック可能かどうか（boolean）
- `className`: 追加の CSS クラス（文字列）
